## 📁 함께 제공되는 파일

```
/repo-root/
├── data/
│   └── zimcarry_data.json   ← 핵심 데이터 (모든 분석 결과)
├── HANDOFF.md               ← 이 파일
└── (사이트 코드는 여기서 만들면 됨)
```

**zimcarry_data.json**의 구조:
- `meta`: 메타 정보 (회사명, 데이터 기간, 단위)
- `company`: 회사 기본 정보 + 투자 라운드 + 마일스톤
- `financials`: 5개년 재무 데이터 (BS/IS/현금흐름)
- `ratios`: 재무 비율 (성장성/수익성/안정성/활동성/투자회수)
- `income_items`: 손익계산서 항목별 분석 (학습노트 + 투자관점 포함)
- `balance_items`: 재무상태표 항목별 분석
- `business_structure`: 35개 거점 + 7개 사업 카테고리
- `dashboard`: 대시보드 카드용 데이터 (5대 카테고리 KPI)

## 🖥️ 사이트 구조 — 4개 탭

### 탭 1: 대시보드 (`/`)

**목적**: 한 화면에 짐캐리 재무 상태 요약

**구성:**
1. **상단 KPI 카드 6개** (가로 한 줄)
   - 매출 76억 / 영업이익 -6.2억 / 순이익 -6.4억
   - 자본 7.95억 / 현금 10.1억 / Runway 16.2개월
   - 데이터: `dashboard.top_kpis`

2. **5대 카테고리 패널** (세로 5개 또는 2x3 그리드)
   - 성장성 🟢 / 수익성 🟡 / 안정성 🔴 / 활동성 🟢 / 현금흐름 🔴
   - 각 카테고리당 KPI 3~4개 + 신호등 + 한 줄 결론
   - 데이터: `dashboard.categories`

3. **종합 평가 박스**
   - "🟡 전환기" + 시나리오 (낙관/기본/비관)
   - 데이터: `dashboard.overall_assessment`

4. **5년 추이 차트 (선택)**
   - 매출 + 영업이익 + FCF 라인차트
   - 데이터: `financials.income_statement.revenue` 등

---

### 탭 2: 재무상태표 (`/balance-sheet`)

**목적**: 자산·부채·자본 항목 디테일 + 학습 노트

**구성:**
1. **3섹션 카드** (자산 🟦 / 부채 🟥 / 자본 🟩)
   - 각 섹션 합계 + 비중 (도넛 차트)
   - 5년 추이 미니 차트

2. **항목 테이블** (섹션별)
   - 컬럼: 항목명 | 5년 수치 | YoY | 비중 | 추이 | (신호등)
   - 행 클릭 시 → 모달 열림

3. **항목 클릭 모달**
   - 5년 추이 라인차트
   - 학습 노트 (개념 설명)
   - 투자 관점 (시사점)
   - 데이터: `balance_items[].items[]`

4. **투자 유치 분석 박스**
   - 누적 투자금 4,617백만 + 라운드별 증가
   - 자본잠식 사이클 시각화 (2021, 2024)

---

### 탭 3: 손익계산서 (`/income-statement`)

**목적**: 매출·비용 항목 디테일 + 비용 구조 분석

**구성:**
1. **흐름도 (Waterfall)**
   - 매출 → 매출원가 → 매출총이익 → 판관비 → 영업이익 → 순이익
   - 2025년 기준 시각화

2. **판관비 구성 (2025년)**
   - 도넛/스택바 차트
   - 인건비 56% / 임차료 22% / 플랫폼 10% / 운송 3% / 기타 9%
   - 비용유형 태그별 색상 구분

3. **5년 추이**
   - 매출 vs 판관비 vs 영업이익 라인차트
   - 매출 대비 % 추이도 보여주기

4. **항목 테이블 + 모달**
   - 재무상태표 탭과 동일 구조
   - 데이터: `income_items[].items[]`
   - **태그 색상**: 인건비/임차료/플랫폼/운송/시설운영/마케팅/금융/매출/손익/기타

---

### 탭 4: 현금흐름 & 투자지표 (`/cash-flow`)

**목적**: OCF/CAPEX/FCF/EBITDA 등 투자 평가 핵심 지표

**구성:**
1. **EBITDA 흐름**
   - 영업이익 + 감가상각 + 무형자산상각 = EBITDA
   - 5년 막대차트

2. **OCF 역산**
   - 순이익 + 비현금비용 - NWC 변화 = OCF
   - 컴포넌트 시각화

3. **CAPEX 분해**
   - 유형/무형/투자자산 증가분 + 감가
   - 어디에 투자했는지 (특히 개발비 9.7억 강조)

4. **FCF 트렌드**
   - OCF - CAPEX = FCF
   - 5년 라인차트 (음수 영역 강조)
   - 누적 FCF -35억

5. **추가 지표**
   - 이자보상배율 (영업이익 / 이자비용)
   - Runway (현금 / -월간 FCF)
   - 차입금 / EBITDA

## 🔑 핵심 인사이트 (사이트 본문에 활용)

### 짐캐리의 5가지 핵심 진실

1. **"PMF는 검증됐다"** — 매출 7년간 11.5배 성장. 시장 수용 확인
2. **"인건비 + 임차료가 매출의 82%"** — 구조적 적자 원인. 단 매출 성장이 비용 압도하며 개선 중
3. **"자본잠식 2회"** — 2021년 -36, 2024년 19백만으로 거의 잠식. 증자로 회생
4. **"투자금 83% 소진"** — 누적 46억 받아 38억 까먹음. 외부 자금 의존 사이클
5. **"BEP 가시권"** — 영업이익률 -33% → -8% 빠른 개선. EBITDA 마진 -5%로 양수 임박

### 투자 베팅 포인트

> **흑자 전환 시점**이 핵심. 2026~2027년 가능성 있으나, 인천공항·SRT 신규 비용으로 단기 적자 확대 가능. 2027~2028년이 진짜 분기점.

### IR 2024 한계점 (사이트 주석으로 활용)

- IR p.23: 특송 사업부 1,512백만 vs IR p.17: 빅5 합계 2,019백만 → **IR 자체 모순**
- 2025년 매출 계획 107억 vs 실제 76억 → **달성률 71%**
- 해외 진출 7개국 11개 도시 → 실제 일본·베트남 6개 매장만

## ⚠️ 주의사항

1. **숫자 단위 일관성**
   - JSON: 백만원 단위 (단, `억` 표시 시 ÷100)
   - 화면: "76억" / "1,420백만원" 등 가독성 우선
   - `lib/format.ts`에 포맷 함수 통일

2. **저작권/개인정보**
   - DART 공시 자료는 공개 정보
   - IR 2024 자료는 외부 유출 주의 (사이트 비공개 권한 설정 권장)

3. **신호등 임계값**
   - JSON에 이미 평가 결과 있음 (`signal: "green"`)
   - 동적 계산 원하면 `lib/signal.ts`에서 임계값 함수화
